Observations from the data

Two platforms in this group achieved the maximum R: 20/20 score — Browserbase and Runner H. Both document zero CVEs in the trailing twelve months. The gap between these two emerges in the D (Data Conduct) dimension: Browserbase publishes an explicit no-training commitment covering all browser data; Runner H's privacy policy (dated 2024.11.19) states that AI is trained on user Inputs, Actions, and Outputs, scored D: 5/15 under the framework.

Skyvern is the only browser agent in the index with standing SOC 2 Type II and HIPAA compliance. Its multi-agent architecture includes a dedicated Password Agent and credential vault integrations for 1Password, Bitwarden, and Azure Key Vault. Stagehand holds the highest V: 16/20 among browser agents, reflecting its TypeScript MIT codebase with verified signed releases and transparent commit history.

Browser Use is the only platform in this comparison with a recorded Critical CVE (CVE-2025-47241, CVSS 9.3) in the trailing twelve months. The vulnerability — allowed_domains bypass via HTTP authentication username — was patched in v0.1.45. The platform holds Y Combinator W25 membership and a $17M seed round led by Felicis Ventures.

Pattern notes

Substance vs Optics
Browserbase
R: 20/20 reflects documented isolation controls (VM-per-session) and audited compliance, but T: 3/10 reflects gated Trust Center access and the absence of public SECURITY.md in the Browserbase and Stagehand GitHub repositories.
Research vs Governance
Runner H
$220M seed round, two arxiv papers, and state-of-the-art benchmarks (WebVoyager 97.1%) coexist with a privacy policy stating that AI is trained on user inputs. "GDPR compliant" does not imply absence of training on customer data.
Certification Leader
Skyvern
SOC 2 Type II and HIPAA compliance are both documented. Published case studies cover insurance (Geico), finance, healthcare, and government verticals. D: 4/15 reflects undocumented sub-processor list and retention periods.
Transparency Leader
Stagehand
MIT license, TypeScript, verified signed releases, 21,700+ stars. The observe() primitive provides explicit HITL action preview. 1Password Secure Agentic Autofill integration delivers HITL credential handling with JIT injection.

Methodology note

*Stagehand inherits SOC 2 Type 1 + HIPAA from its Browserbase Inc. parent when used via Browserbase Cloud. The MIT-licensed SDK itself has no standalone certification. Browser Use holds no independent compliance certifications as of evaluation date.

VERDICT evaluates using publicly available documentation, CVE databases, and vendor-published security pages. No vendor sponsorship, no paid certification, no pre-publication sharing with vendors. Scores reflect what is documented and what is not. Framework v0.3.1.
